# SMT files for SPARKNaCl, re-generated for Z3

This directory contains the SMT files for verification of SPARKNaCl,
but generated by the Why3 VCG using its own specific driver for Z3,
which specializes the output for Z3 alone.

There are 5998 files in this directory.

Contrast these files with those generated for cvc5 in `../why3-cvc5`.

## Result

Running Z3 4.15.3 with 5s timeout on these files yields:

```
unsat:   5747
timeout:  251
total:   5998
```

Entertainingly, this result is *worse* than what we get running Z3 on the SMT files
generated for cvc5.
